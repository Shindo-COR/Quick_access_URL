# URL Command Center 拡張機能 使用方法

## 0. 拡張機能の紹介
URL Command Center は、日々の開発業務・運用業務・調査作業で使う URL を
瞬時に呼び出せる開発者向けショートカット拡張機能です。

この拡張は「単なるブックマーク」ではありません。
開発者の認知負荷を削減する作業オペレーションツールです。

「どの Jenkins だっけ？」
「Backlog URLどこ？」
「この案件の管理画面、ブックマーク探すのだるい…」

そんな“思考の摩擦”をゼロにするために生まれました。

## 1. 拡張機能を開く

1. Chrome のツールバーから **URL Command Center アイコン** をクリック  
2. 拡張機能のメイン画面が表示されます  

> ※初回はデフォルトのタブとボタンが表示されます

---

## 2. マイセットの操作

### マイセットを選択する
- 画面左側のマイセットのタブをクリックすると、右側にそのマイセットに紐づくボタンが表示されます

### マイセットを並び替える
- マイセットのタブをドラッグ＆ドロップすることで、自由に順序を変更可能

### マイセットの追加
1. 左側の「＋」ボタンをクリック  
2. 新しいタブが作成されます（タブは最大 20つ まで）
3. マイセット名を入力します。この時「CCA-1728」など課題キーを入力するとAPIから自動的に情報を取得し、自動的にボタンがセットされます
   ※初期値はココアのバックログで設定されています。defaultconfig.jsから変更することができます
   ※事前にdefaultconfig.jsにAPIキーを設定する必要があります。
   パス：「addon_URL Command Center\URL Command Center\files\config\defaultconfig.js」
   ■設定方法
   1. ファイルを開き以下の項目を設定する
   2. backlogApi: {
		space: "cor",
		apiKey: ""  ⇐ここにご自身のapiキーを入力。  ※バックログ＞自身のアイコン＞個人設定＞APIからご自身のAPIを登録してください。
	},
   3. 拡張機能を更新する

---

## 3. ボタンの操作

### ボタンをクリック
- URL が設定されているボタンをクリックすると、新しいタブでその URL が開きます

---

## 4. 編集画面の使い方

1. タブ名を変更する  
2. 必要に応じてボタンを編集、追加、削除する  
3. 「編集を閉じる」をクリックすると即時反映＆保存  

### タブの複製・削除
- **複製**：同じ内容のタブをコピー（最大 20 タブまで）  
- **削除**：タブを削除（最低 1 タブは必須）  

### 初期化
- 「初期化」ボタンでそのタブをデフォルト設定に戻せます  
- デフォルト設定は `defaultconfig.js` に基づく内容  

### ボタンの追加
1. 編集画面で「＋ URL追加」をクリック  
2. 新しいボタン行が追加され、名前・URL・色を設定可能
  
### バックログ自動入力
### ※事前にdefaultconfig.jsにAPIキーを設定する必要があります。詳しくはdefaultconfig.jsの説明と設定方法.mdをご確認ください。
1. バックログの種類を選択（初期値は「ココア」「リラクジョブ」「内部課題」です。defaultconfig.jsから変更できます） 
2. 数字を入力 
3. Enterを押す
4. バックログのラベルとURLが自動的に入力されたボタンが出現します

## ５. 設定画面の使い方

### UI配置
- **閉じる/設定ボタンを上に表示**
- チェックボックス選択で上に配置します。素早く設定ボタンをクリックしたい時に最適です
- チェックボックス未選択で下に配置します。よりURLボタンが見やすくなります

### ダークモード設定
- **ダークモード**：チェックボックス選択で背景色と文字色を設定可能  
- **背景色**：背景色を選択。※マイセットの箇所は変更されません
- **文字色**：文字の色を選択。※URLボタンは変更されません

### エフェクト
- チェックボックス選択でURLボタンホバー時に虹色になります。

### 設定データ
- **初期設定にリセット**：インストール時の初期設定にリセットします  
- **設定をエクスポート**：マイセットやダークモードの色等の設定をJSONファイルで出力します
- **設定をインポート**：エクスポートしたJSONファイルを取り込みます。

---

## ６. 注意点

- 編集内容は即時保存されます  
- ブックマークとは異なり、**画面上で直感的にリンク管理が可能**  
- 大量のお気に入りから探す必要がありません

---

## ７. 使い方など

-  案件でよく見るドキュメントやページを設定しておく
-  「URL Command Center」 で媒体にアクセス　→　「EKI Env Switch」で環境を入れ替える

## ８. よくある質問

### Q. タブやボタンを間違って削除した
A. 初期化ボタンを押すと `defaultconfig.js` の初期値に戻ります

### Q. 最大タブ数は？
A. 現在 20 タブまで作成可能です

### Q. ボタンの色は変更できる？
A. 編集画面でカラー選択、またはプルダウンからカラー編集可能です

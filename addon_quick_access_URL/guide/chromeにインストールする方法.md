# quick_access_URL 拡張機能 インストール方法
※Chromeだけでなく Edge でも同様にインストール可能です

---

## 1. 初めてインストールする・新しいバージョンをインストールする場合

1. Chrome 右上の **ケバブメニュー（三点リーダー）** → **拡張機能** → **拡張機能を管理** を開きます  
2. 左上にある **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリック  
3. **quick_access_URL フォルダ** を選択して読み込みます

> ⚠️ 注意
> - 新しいバージョンをインストールする場合は、先に古いものを削除してください  
>   <!-- 例: ![削除ボタン](https://gyazo.com/ae5e2382d4d26791ec70a0e504417d93) -->
> - 「パッケージ化されていない拡張機能を読み込む」が表示されない場合は、右上の **デベロッパーモード** をオンにしてください  
>   <!-- 例: ![パッケージ化されていない拡張機能](https://gyazo.com/0f8a35380863c413fdf7c5e9c0c9fb94) -->

4. Chrome 右上の拡張機能アイコンから **quick_access_URL** をピン留めすると便利です  
   <!-- 例: ![拡張機能ピン留め](https://gyazo.com/e188e789fb29ad20d953c35a04eeeb24) -->

5. （任意）江口さん作成の拡張機能 **EKI Env Switch** もピン留めしておくと、隣で同時に操作可能です  
   <!-- 例: ![EKI Env Switch](https://gyazo.com/c303438dd46d933252e73eeb2388ca71) -->

---

## 2. 更新する場合（バージョンアップ）

1. 「quick_access_URL」フォルダ内のファイルを新しいバージョンに差し替えます
2. Chrome 右上の **ケバブメニュー（三点リーダー） → 拡張機能 → 拡張機能を管理** から  
   「quick_access_URL」の枠の右下にある **更新ボタン** を押します  
   <!-- 例: ![更新ボタン](https://gyazo.com/9f169a44214a2e6535b538e53bfec5a5) -->
3. 更新完了です。ピン留め済みの場合は再度操作不要です
